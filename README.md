# sistema_de_provas

Estrutura de projeto com separação backend e frontend:

- `sistema/backend` - API Node.js + TypeScript + Express
- `sistema/frontend` - Next.js + React + TypeScript

## Backend

`cd sistema/backend`
`npm install`
`npm run dev`

## Frontend

`cd sistema/frontend`
`npm install`
`npm run dev`
